July 2024 - August 2024

NBA Game Predictor GitHub

Skills: Pandas,Scikit-Learn,Web Scraping, Classification

As a basketball enthusiast, I couldn't resist the opportunity to put my data analysis skills to the test by building a model to predict NBA game outcomes. I started by web scraping comprehensive datasets on team and player performance metrics which includes FG%, and 3P%, then preprocessed and clean the data to ensure the data is effective for machine learning. Finally, developed a predictive algorithm using the RidgeClassification model from Scikit-Learn that could forecast likely winners based on the data from previous seasons. The predictive model effectively predicted the correct outcome with an accuracy score of 78%.

2014-2015

Shopper E-Commerce GitHub

Cambridge University

For this project, I designed and developed a responsive, React-based web application from scratch using the MERN (MongoDB, Express.js, React, Node.js) technology stack. I put careful thought into the overall architecture and user experience, seeking to create an intuitive and visually appealing e-commerce platform. On the backend, I implemented robust server-side components, including APIs and database models, using Node.js and Express.js. This allowed me to build out comprehensive functionality for managing the product catalog and order processing. A key aspect of the project was developing a secure and feature-rich admin dashboard. This dashboard enables authorized users to easily add, edit, and remove products in the e-commerce catalog, with all data stored securely in a MongoDB Atlas database. I paid close attention to access controls and data integrity to ensure the admin tools were reliable and protected.
